Whisper — Setup

Purpose: Speech‑to‑text for voice memos, meeting notes, and subtitles. Uses local venv and wrapper scripts.

Skill

Intermediate

Estimated time

15–45 minutes (model download time varies)

Setup

  1. Install or recreate the virtual environment: 
    bash t-whisper/install.sh
  2. Transcribe an audio file with the wrapper: 
    ./t-whisper/transcribe.sh audio.mp3
  3. Adjust model and format options as needed (small/medium/large; output formats: txt, srt, vtt).

Verification

Check /root/.openclaw/workspace/x-temp/whisper-output/ for output files and confirm the transcript looks correct.

Troubleshooting

  • Model download fails: ensure network access and enough disk space.
  • GPU issues: confirm CUDA drivers if using GPU acceleration.
