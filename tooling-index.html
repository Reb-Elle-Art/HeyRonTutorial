Tooling Index — Pick What You Need

These are optional. You don't need every tool. Start with what solves a real problem.

The philosophy: Use what serves you. Skip what doesn't. Be curious about the rest.

Essential Tools

These cover common needs. Most people start here.

Git & GitHub
Time: 10-15 minutes • Skill: Beginner
Purpose: Backup, version control, cloud storage for your work

What it lets you do:
  • Save files securely in the cloud
  • Keep history of changes
  • Access from any device
  • Give your agent a permanent home for your work

Communication Tools

Discord
Time: 5-10 minutes • Skill: Beginner
Purpose: Chat with your agent from phone, computer, or web

What it lets you do:
  • Message your agent anytime, anywhere
  • Get notifications on your phone
  • Share files and documents
  • Set up team channels for alerts

How to Use This Index

  1. Pick one tool that solves a real problem you have right now
  2. Read that page — follow the steps
  3. Test it with your agent
  4. Come back for another tool when you need it

Don't try to set up everything at once. One tool, mastered, is better than five tools half-configured.

If You're Stuck

Ask your agent: "Help me set up [tool name]"

Your agent can walk you through any of these step-by-step.
