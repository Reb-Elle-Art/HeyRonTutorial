here.now — Setup

Purpose: Publish static files and folders to a quick shareable URL. Good for prototypes and previews.

Skill

Beginner

Estimated time

5–10 minutes (requires API key for permanent artifacts)

Setup

  1. Ensure API key is in .env as HERENOW_API_KEY if you want permanent artifacts.
  2. Publish a directory: 
    /root/.claude/skills/here-now/scripts/publish.sh ./my-site --client openclaw
  3. Or publish a single file: 
    /root/.claude/skills/here-now/scripts/publish.sh ./document.pdf --client openclaw

Verification

Script outputs a public URL. If anonymous, artifacts expire in 24 hours. With API key, artifacts are permanent.

Troubleshooting

  • Missing API key: anonymous artifacts expire; check .herenow/state.json for published slug info.
