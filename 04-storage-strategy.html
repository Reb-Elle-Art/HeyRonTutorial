Storage Strategy_

Where things live matters. Your agent needs to know where to find persistent data and how to handle backups.

Memory Layers

  • Session memory: What's in the current chat
  • File memory: What's in the workspace
  • Curated memory: Long-term context (SOUL.md, USER.md, MEMORY.md)

Backups

Your agent can push to GitHub. Set up a backup repo and use it for:

  • End-of-session snapshots
  • Before major changes
  • Archive of completed projects
Rule: Never auto-push to public repos without explicit confirmation.
