Troubleshooting_

When things go sideways, here's a systematic approach.

Common Issues

  • File not found: Check the path and ensure it's inside the workspace.
  • Permission denied: Verify file permissions and that the agent has read/write access.
  • Publish fails: Confirm the target directory is clean and no unsaved changes exist.

Diagnostic Steps

  1. Ask the agent to list the workspace contents.
  2. Check the .herenow/state.json for recent actions.
  3. Review the AGENTS.md and HEARTBEAT.md for any required context refresh.
If a step fails, ask for a short error report and then retry with stricter constraints.
